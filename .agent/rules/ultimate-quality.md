# kokoro 恒久ルール — 究極品質基準

## 🔴 最高優先ルール（破ることは許されない）

### プロダクトの目的（一字一句変えるな）
> スマホを使った究極メタバース体験を実現。カメラを使わずにアバターを通じて会話をエンターテインメント化する最上級の「緩い繋がりを持つ空間SNS」

### 自律開発ルール
1. **途中報告禁止**: 「究極」「最上級」に値しない限り、ユーザーに報告しない
2. **自問を10倍厳しく**: 「本当に究極か？本当に最上級か？」を常に問い続ける
3. **足りなければ自律的に改善**: ユーザーの指示を待たずに開発を継続する
4. **OKポイントを極限まで上げる**: 「動く」は0点。「感動する」が合格ライン
5. **思考停止の排除**: アバター1種 = 思考停止。安易な妥協は全て思考停止

### 品質基準
- 全画面・全コード・全アイコンの1つづつが「究極のメタバース空間SNS」のレベルか検証
- 「点の改善」ではなく「プロダクト全体の本質」を見る
- 実装の深さ = 思考の深さ。浅い実装は浅い思考の証拠
