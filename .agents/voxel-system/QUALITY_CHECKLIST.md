# ボクセルアセット品質チェックリスト

> このチェックリストは新規アセット追加時・テーマ追加時に必ず確認する。
> NORTH_STAR見本と判別不可能な品質を保証するための基準。

## 1. 環境 (AjitRoom)

### 壁
- [ ] 上部(明色) + 下部(腰壁 HSL offset: 彩度+0.12, 明度-0.22)の2色分割
- [ ] キャップレール(腰壁上端の帯状モールディング)存在
- [ ] 巾木(床際のモールディング)存在
- [ ] 天井モールディング存在
- [ ] **壁面の40%以上** が棚・フレーム・小物で覆われている

### 床
- [ ] 個別の木の板(boxGeometry, 板幅0.42, 隙間0.03)
- [ ] 交互の板色(100% / 92%)で木目感

### ドア
- [ ] ThemeDoorのz位置: `-ROOM_D / 2 + 0.2` (壁前面から0.12前方)
- [ ] ドア本体の色がテーマに調和する暖色
- [ ] パネル装飾 or アーチ or ハッチのスタイル適用
- [ ] ドアノブ存在

### 天井
- [ ] シーリングライトの物理メッシュ(ベースプレート+シェード+発光球)

## 2. 壁装飾 (ThemeArchitecture)

### 棚
- [ ] Back Wallに**最低2段の棚**(WallShelfWithItems)
- [ ] 各棚に**4個以上**のアイテム(テーマカラー)
- [ ] 棚の色がテーマの木目色と調和

### 壁掛け
- [ ] **最低1つの壁掛けフレーム**(WallFrame)
- [ ] フレームの内側色がテーマアクセント色

### ペンダントライト
- [ ] **最低2つ**のPendantCeilingLight
- [ ] テーマに合ったshape(dome/lantern/industrial/cage/crystal)

## 3. アバター (VoxelAvatars)

- [ ] InstancedMesh方式の角丸BoxGeometry
- [ ] blockScale=0.93 でブロック間の微細な隙間
- [ ] ほっぺ(#FFB6C1のピンクブロック)
- [ ] 目のハイライト(白い小ブロック)

## 4. 家具 (VoxelFurniture)

- [ ] NoisyBox/NoisyCylinder/NoisySphereの最低10パーツ以上
- [ ] bevel値が0.008〜0.02
- [ ] lightnessSpread: 0.06〜0.15(微妙な色むら)
- [ ] テーマカラーパレットと調和

## 5. テーマ装飾 (ThemeDecorations)

- [ ] VoxelGrid系アセット(VoxelCrystal, VoxelRock等)の使用
- [ ] **テーマ固有のカラーブロック**(共通カラーではなくテーマカラー)
- [ ] 小物の配置が意図的(グループ化、棚上配置)
- [ ] 散乱小物は最大5個以内

## 6. 新規アセット追加手順

1. NORTH_STAR見本との比較画像を撮影
2. 差異リスト作成(マクロ+ミクロ)
3. このチェックリスト全項目を確認
4. ブラウザスクリーンショットで検証
5. デプロイ後に再確認
