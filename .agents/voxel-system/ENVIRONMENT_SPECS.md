# cocoro 環境テーマ — 徹底比較分析書

> NORTH_STAR_5(ペンギン暖色部屋)とNORTH_STAR_6(宇宙ステーション)の
> 2画像を1要素ずつ比較し、「共通文法」と「テーマ固有語彙」を抽出する。
> このルールに従えば、新しいテーマでも同じブランドの部屋を生成できる。

## 2. 共通文法（テーマを問わず絶対に含むもの）

### 2.1 空間レイアウト公式

```
=== 部屋の絶対構造 ===

         [左壁: 大型絵画/ボード]
              │
    [植物]    │
    [植物]    │   [丸コンテナ]   [キャラ]   [デスク+PC/モニタ]
              │                              [椅子]
         ─────┤
              │          [奥壁]
              │  [棚+フィギュア]  [ドア(開)]  [棚2段+文字ブロック]
              │                              [スイッチ]
──────────────┘

    [ブロック積み木]  [ポーション]  [本(開)]  [キューブ散乱]

         === 手前（カメラ寄り）===
```

### 2.2 12の必須要素チェックリスト

新テーマを作るとき、以下12要素をすべて含めること:

1. ☐ **大型壁面装飾** — 左壁の上半分、壁面積30%、テーマ世界観を表現するフレーム付きコンテンツ
2. ☐ **棚（2-3段）** — 奥壁のドア両脇、ミニフィギュアとブロック文字を載せる
3. ☐ **ドア（開放）** — 奥壁の右寄り、テーマ素材で表現、必ず開いている
4. ☐ **天井照明** — 暖色、中央1つ
5. ☐ **デスク＋PC/モニタ** — 右壁前、椅子付き
6. ☐ **丸型コンテナ** — 中央、メインキャラの横
7. ☐ **植物2-3鉢** — 左壁前、高さを変えて並べる
8. ☐ **カラフルブロック積み木** — 床の左手前、5-6個散乱
9. ☐ **開いた本** — 床の右手前、1-2冊
10. ☐ **小瓶/ポーション** — 床の中央手前
11. ☐ **散乱キューブ** — 単独の2×2ブロック1-2個
12. ☐ **壁のテクスチャ** — 単色平面禁止、テーマに応じた模様

### 2.3 共通カラー原則

| 原則 | 説明 |
|------|------|
| 暖色照明は不変 | テーマが「宇宙」でも「深海」でも、メイン照明は暖色系 |
| 壁は2トーン | 上部=明、下部=暗の2トーン or テクスチャで分節 |
| 床は反復パターン | 木目交互 or タイル交互 or リベット付き金属 |
| アクセント色は3箇所以上 | 壁装飾+小物+植物で点在させる |
| 黒は使わない | 最も暗い色でも #1A1A1A、純黒#000000は禁止 |

## 4. 小物の精密ブロック仕様

### 4.1 カラフルブロック積み木（共通必須）

参考画像で**両方に**完全に同じスタイルで存在するため、最重要共通小物:

```
構成: 5-8個の2×2×2ブロック
配色（固定5色）:
  - 赤:   #E74C3C
  - 青:   #3498DB
  - 黄:   #F1C40F
  - 緑:   #2ECC71
  - 紫:   #9B59B6
配置ルール:
  - 3-4個は積み上げ（L字やT字に）
  - 2-3個は転がった状態（単独配置）
  - 必ず床の左手前エリア
  - 1個はメインの山から少し離して
シャドウ:
  - 各色の0.75明度を底面1ブロックに
```

### 4.2 開いた本（共通必須）

```
構成: V字型に開いた2ページ + 背表紙
見開きサイズ: 6×1×8（開いた状態）
ページ色: #FFF8DC (ライトクリーム)
表紙色: テーマ色（ペンギン=#8B0000暗赤、宇宙=#654321茶）
ページテクスチャ:
  - 左ページ: 横線テキスト風 = 1ドット間隔で#CCC ライン
  - 右ページ: 小さなピクセルアート(3×3の絵)
  ペンギン部屋: 山と木のピクセルアート(緑+茶)
  宇宙部屋: 回路図風(シアンのドットライン)
背表紙接地: V字の底辺が床に接する
```

## 5. ライティング精密仕様

### 5.1 共通ライティング設定

```
// メインアンビエント
ambientLight: { color: #FFF5E6, intensity: 0.4 }

// 天井ポイントライト
pointLight: {
  position: [center_x, wall_height, center_z],
  color: #FFF8E1,
  intensity: 1.2,
  distance: wall_height * 3,
  castShadow: true
}

// 環境反射（柔らかさの源）
hemisphereLight: {
  skyColor: #FFF0CC,      // 暖色の空
  groundColor: #5C4033,    // 地面の暗色反射
  intensity: 0.3
}
```

### 5.2 テーマ別 アクセント光

```
ペンギン部屋（暖色系）:
  追加光なし（メイン照明のみで十分）

宇宙ステーション（SF系）:
  accentLight1: { // モニタの光
    type: 'pointLight',
    color: #00BFFF,
    intensity: 0.3,
    position: [desk_x, desk_y + 3, desk_z]
  }
  accentLight2: { // クリスタル発光
    type: 'pointLight',
    color: #7c3aed,
    intensity: 0.2,
    position: [container_x, container_y + 5, container_z]
  }

変換ルール:
  暖色テーマ → 追加光は暖色orなし
  SFテーマ → シアン/ネオン系のポイントライト
  自然テーマ → 緑系の微かな発光（蛍/キノコ）
  火山テーマ → 赤/オレンジの溶岩発光
```
