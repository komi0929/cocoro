# cocoro 環境・部屋・小物 — 精密仕様書

> NORTH_STAR_1(ツリーハウス)、NORTH_STAR_4(ペンギンの部屋)、
> ref_room_space(宇宙ステーション)、ref_room_rabbit(ウサギの地下室)
> の4つの部屋シーンから抽出した、環境生成の絶対ルール。

---

## 1. 部屋の構造原則（全テーマ共通）

### 1.1 アイソメトリック・ジオラマ構図

全参考画像に共通する構図:
```
        ┌─────────────────────┐ ← 天井（見えない or 照明のみ）
        │ [壁面左]   [壁面奥]  │
        │  棚+小物   絵画+棚   │ ← 壁面: 情報密度が非常に高い
        │   │          │       │
        │   └──────────┘       │
        │    ╔══════════╗      │ ← ドア（1つ、右寄り or 奥中央）
        │    ║  床面     ║      │
        │    ║ 小物散布  ║      │ ← 床: 30-40%が小物で埋まっている
        │    ╚══════════╝      │
        └─────────────────────┘
        ── 段差/テレイン ──       ← 床の端は段差的に数層落ちる（島感）
```

### 1.2 壁面の構成ルール

**壁面は「情報密度の高いディスプレイ」**として扱う。空白の壁面は許されない。

| 要素 | 説明 | 参考画像の具体例 |
|------|------|------------------|
| 大型絵画 | 壁面積の25-30%を占めるフレーム付き | ペンギン部屋:恐竜ピクセルアート、ウサギ部屋:ネットワーク地図 |
| 棚（2-3段） | 壁に固定、上にフィギュア・本・小物 | 全部屋共通:棚の上にミニキャラ・ブロック文字 |
| ドア | 木or金属、取っ手+窓付き、開放状態 | ペンギン:木目ドア開放、ウサギ:金属ドア開放 |
| 壁の色分け | 上半分=明色、下半分=暗トーンの腰壁 | ペンギン:上=水色/下=ティール |
| 配管/レール | テーマに応じた壁面ディテール | 宇宙:金属パイプ、ウサギ:レンガ目地 |
| 照明スイッチ | 壁面の小さなスイッチプレート | ペンギン:右壁に白い四角 |

### 1.3 床面の構成ルール

**床は「生活感の宝庫」**。

| 要素 | ルール |
|------|--------|
| 床材 | 木目パターン: 2-3ブロック幅の板×交互色(明/暗ブラウン) |
| 小物散布 | 床面積の30-40%に小物を配置 |
| 散らかし | ブロック(カラフル2×2)、本(開いた本)、カップ、道具の残骸 |
| 動線 | キャラクターの周囲は少し空ける |
| 端の段差 | 床端は2-3層の段差的テレイン（島感を演出） |

### 1.4 照明ルール

| 要素 | ルール |
|------|--------|
| メイン光源 | 天井中央のシーリングライト（暖色 #FFF8E1、円形or四角） |
| 光の色温度 | 暖色基調（3000-4000K相当） |
| 影 | 柔らかいコンタクトシャドウ（contactShadows） |
| アクセント光 | テーマに応じた小型光源（ランタン、モニタの光） |
| 暗い角 | 完全に真っ暗にはしない、反射光で柔らかく |

## 2. 床上小物カタログ（全テーマ共通で使える）

参考画像に繰り返し登場する小物:

### 2.1 カラフルブロック（積み木）
```
構成: 5-8個のランダム配置ブロック
サイズ: 各2×2×2
色: 赤(#E74C3C)、青(#3498DB)、黄(#F1C40F)、緑(#2ECC71)、紫(#9B59B6)
配置: ランダムに散らす（積み重ねor散乱）
```
**全4部屋に登場** — これはテーマを問わない共通小物。

### 2.2 開いた本
```
構成: 見開き2ページ＋表紙
サイズ: 6×1×8（開いた状態）
色: ページ=#FFF8DC(クリーム)、表紙=#8B0000(暗赤) or テーマ色
テクスチャ: ページ上に黒ドットで文字/絵を示唆
```

### 2.3 ポーション/ボトル
```
構成: 小瓶＋コルク
サイズ: 2×4×2
色: 瓶=#B0E0E6(ライトブルー/透明)、コルク=#DEB887
```

### 2.4 カップ＆ソーサー
```
構成: カップ＋皿
カップ: 3×3×3（中空）、白/#FFFFFF
ソーサー: 4×1×4、白
ドリンク: 上面1ドット暗色(#6F4E37)
```

### 2.5 キャンドル/ランタン
```
構成: 台座＋炎
サイズ: 2×3×2
色: 台座=ゴールド(#B8860B)、炎=オレンジ(#FF8C00)発光
```

### 2.6 歯車/メカパーツ
```
構成: 平らな円形ギア
サイズ: 直径4-5ブロック、高さ1
色: ブラウン/グレー(#8B7355)
※宇宙/地下室テーマに特に多い
```

### 2.7 ミニフィギュア（棚用）
```
構成: 超小型キャラ（3×5×3程度）
色: カラフル（赤/緑/青/黄など各1色）
配置: 棚の上に3-5体並べる
```

## 3. 壁面装飾カタログ

### 3.1 額縁付き絵画（大）
```
フレーム: ウッドブラウン(#8B6914)、幅1ブロック
サイズ: 20×15ブロック（フレーム含む）
内容: テーマに応じたピクセルアート
  - ペンギン部屋: 恐竜（ラプラス風）の海の絵
  - ウサギ部屋: 地下ネットワーク地図
  - 宇宙部屋: 星座チャート
配置: 左壁 or 奥壁の上半分
```

### 3.2 棚（壁掛け）
```
構成: ブラケット2個 + 板
板: 8×1×3、ウッドブラウン
ブラケット: 三角形（2×2の直角三角）、ダークブラウン
配置: 壁の中段〜上段に2-3段
上の小物: フィギュア、ブロック文字、本、瓶
```

### 3.3 ブロック文字
```
構成: アルファベット型2×3×2ブロック（A, B, C等）
色: 各文字ごとに赤/緑/黄/青
配置: 棚の上に3-5文字横並び
```

### 3.4 壁掛け時計
```
構成: 円形フレーム + 文字盤 + 針
直径: 7ブロック程度
色: フレーム=#8B7355(ウッド)、文字盤=#FFFFFF、針=#1A1A1A
```

### 4.4 🌳 ツリーハウス (Treehouse)

**世界観:** 巨木の中に作られた秘密の家
**参考:** NORTH_STAR_1が直接の北極星
**カラーパレット:**
| 用途 | 色 |
|------|-----|
| 壁 | #5C3D1E〜#3D2510(木の幹テクスチャ、縦ストライプ3色) |

**NORTH_STAR_1から読み取れる具体要素:**

- **階段:** 2-3段の小さなステップが床から外へ
- **水場:** 石枠のプール（ライトブルー水面）
- **外部の木:** 背景に細い木が数本（幹=2×2ブラウン、葉=緑の球体/直方体）

## 5. テクスチャパターン辞典

ボクセルで素材感を表現する具体的なパターン:

### 5.1 木目
```
方法: 2色の水平ストライプ交互
色: 明(#A0752E) / 暗(#8B6914)
幅: 各2-3ブロック
方向: 水平（床）or 垂直（壁柱）
```

### 5.2 レンガ
```
方法: 2色の矩形ブロック配置（互い違い）
レンガ色: #9E5B3C
目地色: #C8B9A5
レンガサイズ: 4×2×1
目地幅: 1ブロック
```

### 5.3 金属パネル
```
方法: 均一色面 + リベット点 + 継ぎ目ライン
パネル色: #B0BEC5 or #607D8B
リベット: #90A4AE の1ドット、3-4ブロック間隔
継ぎ目: パネル色より1段暗い1ドット幅のライン
```

### 5.4 石/岩
```
方法: 3色のランダム散布
色: ベース(#5C4033) + ライト(#6B5444) + ダーク(#4A3728)
比率: 60:25:15
パターン: ノイズ的（隣接同色を避ける）
```

### 5.5 苔/ツタ
```
方法: 壁面に1-2ドットのクラスターをランダム配置
色: #228B22(メインの苔緑) + #2E8B57(暗緑)
密度: 壁面ブロックの5-10%
配置: 下部と隅に多く、上部は疎
```

### 5.6 布/ファブリック
```
方法: 2色の微細パターン（チェック or 無地+アクセント）
ベース色: パステル系（ベージュ、ピンク等）
パターン: 2ブロック間隔のドット or 市松模様
```

## 6. スケール関係チャート

すべてのアセットは以下のスケール関係に従う:

```
アバター全高 ≈ 30ブロック（基準）
  │
  ├─ 大型家具（ソファ/ベッド/棚）= アバター全高の 50-80%
  ├─ 中型家具（テーブル/椅子）= アバター全高の 30-50%
  ├─ 小型家具（ランプ/時計）= アバター全高の 15-30%
  ├─ 小物（カップ/本/ブロック）= アバター全高の 5-15%
  │
  ├─ ドア高 = アバター全高の 120-150%
  ├─ 壁高 = アバター全高の 150-200%
  └─ 部屋幅 = アバター全高の 300-400%
```

## 7. 新テーマ追加テンプレート

新しいテーマ（例: 図書館）を追加するとき:

```
【テーマ名】図書館 (Library)

【世界観キーワード】
3-5個のキーワード → 静寂、知識、木と革、暖色照明、天井の高さ

【カラーパレット決定】
- 壁: (テーマの主要壁色)
- 床: (テーマの床材パターン+色)
- アクセント: (ポイントカラー)
- 光源: (照明の色温度)

【必須要素チェック】
□ 大型壁面装飾 1-2個
□ 壁掛け棚 2-3段（ミニフィギュア付き）
□ ドア 1個
□ 照明 1-2個
□ 床上大型オブジェ 1-2個
□ 床上小物 5-10個（カラフルブロック含む）
□ 植物 1-3個

【固有装飾リスト】
テーマ固有の装飾を5-10個リスト

【テクスチャ】
壁面・床面のテクスチャパターンをSection 5辞典から選択

【ライティング】
テーマに応じた光源と色温度
```
