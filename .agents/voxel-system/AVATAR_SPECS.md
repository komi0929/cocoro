# cocoro ボクセルアバター — 精密仕様書

> このファイルのルールに従えば、NORTH_STAR_2の8体を1ドットの狂いもなく再現でき、
> かつ**新しい動物**をこのブランドイメージで生成できる。

---

## 1. 共通テンプレート（buildAvatar）

全アバターはこの骨格に従う。数値は **ブロック数** で記載。

### 1.1 全体プロポーション

```
         耳（種族固有）
    ┌──────────────────┐
    │                  │
    │   頭 (H=12, W=14)│  ← 角丸 cornerR=1
    │  ●       ●      │  ← 目 (3×3 黒 + 白ハイライト左上1dot)
    │      ▼          │  ← 鼻 (2×1 or 2×2)
    │  ⬜ マズル ⬜    │  ← マズル = 白/ベリー色の楕円(rx=4,ry=3)
    │     ─ ─         │  ← 口 (幅4, 1dot高さ)
    ├──────────────────┤
    │   体 (H=10, W=12)│  ← 角丸 cornerR=1
    │ ┃ ベリー白  ┃   │  ← ベリー = 体正面に白/明色矩形(W=8,H=7)
    │ ┃          ┃   │  ← 腕 = ストレート矩形(W=3,H=8)
    │ ┃          ┃   │
    ├─┘          └──┤
    │ ╻            ╻ │  ← 脚(W=4,H=4〜5)
    └─┘            └─┘
```

### 1.2 共通寸法テーブル

| パーツ | 幅W | 高さH | 奥行D | cornerR | 備考 |
|--------|-----|-------|-------|---------|------|
| 頭 | 13-14 | 11-12 | 9-10 | 1 | 最大パーツ |
| 体 | 11-12 | 9-10 | 8-9 | 1 | 頭の85-90%幅 |
| 腕(片) | 2-3 | 7-8 | 2-3 | 0 | ストレート、体から直接接続 |
| 脚(片) | 3-4 | 4-5 | 3-4 | 0 | 脚間隔 legSpacing=1 |
| ベリー | 7-8 | 6-7 | - | - | 体前面に塗る、ベリー色 |

### 1.3 色の方程式

```
メイン色 → cp(base)
  ├─ base: そのまま使う（体・頭・腕・脚の主色）
  ├─ shadow: adjustBrightness(base, 0.75) → 底面・影側に使う
  └─ highlight: adjustBrightness(base, 1.15) → 頂面に使う

ベリー色 → cp(belly)
  └─ base: 白〜クリーム系（マズル・お腹に使う）
```

### 1.4 顔パーツ配置（正面ビュー座標 — headの左下を原点とする）

| パーツ | X位置 | Y位置 | サイズ | 色 |
|--------|-------|-------|--------|-----|
| 左目 | cx - eyeSpacing - 2 | headH × 0.5 | 3×3 | #1A1A1A 黒 |
| 右目 | cx + eyeSpacing | headH × 0.5 | 3×3 | #1A1A1A 黒 |
| ハイライト | 目の左上1ドット | - | 1×1 | #FFFFFF |
| 鼻 | cx - 1 | headH × 0.35 | 2×2 | 種族固有色 |
| 口 | cx中央 | headH × 0.2 | 幅4×高1 | #666666 |
| マズル | cx中央 | headH × 0.35中心 | 楕円(rx=headW×0.3, ry=headH×0.3) | ベリー色 |
| ほっぺ | cx ± 5 | headH × 0.35 | 1×1 | #FFB6C1 ピンク |

**eyeSpacing = 2〜3**（目の内側端から中心線までのブロック数）

## 3. 新動物追加プロンプト（テンプレート）

新しい動物（例: フクロウ）を作るとき、以下のテンプレートに沿って設計する:

```
【動物名】フクロウ (Owl)

【カラーパレット決定】
- メイン: その動物の最も特徴的な色 → #6B4E2E (ダークブラウン)
- シャドウ: メイン × 0.75 明度
- ハイライト: メイン × 1.15 明度
- ベリー: 白〜クリーム系
- アクセント: 種族固有色（くちばし=#FF8C00 等）

【寸法決定】
- 共通テンプレートの範囲内（headW=11〜14, bodyW=10〜12）
- その動物の体型特徴を反映:
  - ふっくら? → bodyW大きめ
  - スリム? → bodyW小さめ
  - 大頭? → headW大きめ

【最大特徴の決定（1つ）】
- フクロウなら「大きな丸い目」
- これが他の動物との差別化ポイント
- buildSpecial or drawFaceExtras で実装

【耳の決定】
- 既存の4タイプから選ぶ or 新型を追加:
  - buildRoundEars: クマ、パンダ、ハムスター
  - buildPointedEars: ネコ、キツネ
  - buildLongEars: ウサギ
  - buildFloppyEars: イヌ

【アイテムの決定】
- その動物と関連する小物1つ
- 右手先に配置

【チェック】
□ 既存8体と並べて同じ世界の住人に見えるか？
□ 色数は5色以内か？
□ 大頭身比率になっているか？
□ ほっぺドットがあるか？
□ 角丸cornerR=1が全パーツに適用されているか？
```

## 4. 家具・小物精密仕様（NORTH_STAR_3分析）

### 4.1 ソファ
- **全体**: ベージュ系ナチュラル、背もたれ3ブロック高、座面2ブロック高
- **脚**: ウッド系4本、各2×2×2
- **クッション**: 座面にパターン入り小クッション（格子柄=2色交互）
- **スケール**: アバターが座れるサイズ（幅=アバター体幅×2.5）

### 4.2 テーブル（サイドテーブル）
- **天板**: 4×4×1 ウッド色、角丸
- **脚**: 4本、各1×3×1
- **上の小物**: カップ＆ソーサー、ランプ等を載せられる

### 4.3 椅子
- **構造**: 背もたれ板(3×5×1) + 座面(4×1×4) + 脚4本
- **色**: ウッドブラウン1色 + 暗色シャドウ

### 4.4 テレビ
- **筐体**: レトロブラウン木目台、上にアンテナ2本（V字）
- **画面**: ダークグレー平面
- **脚**: 4本の小さな脚

### 4.5 冷蔵庫
- **本体**: クリーム色の大きめ矩形（8×14×6程度）
- **取っ手**: シルバー2ドット（上段/下段）
- **分割線**: 上段・下段の境目に1ドット溝

### 4.6 時計（壁掛け）
- **形状**: 丸型ピクセルアート（直径7ブロック程度の円）
- **文字盤**: 白背景 + 黒の12/3/6/9位置マーク + 黒い針2本
- **フレーム**: ブラウン1ドット幅の円

### 4.7 花瓶＋花
- **花瓶**: 白い台形（下4幅→上6幅、高さ5）
- **花**: 赤い花びら(2×2) + 黄色中心(1×1) + 緑の茎と葉

### 4.8 植木鉢
- **鉢**: テラコッタ色の台形
- **植物**: 緑2色のサボテン or 多肉

### 4.9 本（積み上げ）
- **構成**: 3冊重ね、各色違い（赤/緑/紫等）
- **サイズ**: 各 3×1×4 程度

### 4.10 毛糸玉
- **形状**: ターコイズ球体r=3
- **テクスチャ**: 溝パターン（3ドット間隔でダーク色ライン）
